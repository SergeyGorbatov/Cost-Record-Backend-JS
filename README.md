# ***Описание проекта***
Серверная часть проекта "Учёт расходов".
Возможно создавать, удалять, изменять и отмечать мои расходы. 
Использовал MongoDB в качестве базы данных. Также написал небольшую валидацию.

***Технологии:***
+ ***Express*** 
+ ***mongoose***
+ ***MongoDB***
+ ***express-validator***
